# Features ✨

- **🎧 Music Player Controls**: Play, pause, skip, and navigate through songs.
- **💡 Dynamic Lighting Bars**: Bars change colors and height based on the music's tempo.
- **🌙 Theme Toggle**: Switch between light and dark modes.
- **🖼 Wallpaper Selection**: Choose a wallpaper from the popup for background.
- **📱 Responsive Design**: The player adapts to various screen sizes and offers a smooth user experience.

# Technologies Used 💻

- **HTML5**: Structure and content of the webpage.
- **CSS3**: Styling and animations.
- **JavaScript**: Logic for music player controls, lighting bar animation, and interactivity.
- **Swiper**: A responsive slider for album cover images.
- **Font Awesome**: Icons for controls and theme toggle.

# Usage 🚀
-**🎶 Play/Pause:** Click the play/pause button to control the song.

-**⏩ Skip Songs:** Use the forward and backward buttons to navigate through the playlist.

-**💡 Dynamic Lighting Bars:** The lighting bars will animate based on the current song, changing color and height.

-**🌞 Theme Toggle:** Click the sun/moon icon in the top-right to toggle between light and dark modes.

-**🖼 Wallpaper Popup:** Click the wallpaper icon to choose a background image.

# Customization 🖌
To add more songs, update the songs array in script.js with the song's title, name, source file path, and lighting bar color.
To change the color scheme, update the color values in the colors array in script.js.
Modify the CSS for design adjustments like sizes and positioning.
